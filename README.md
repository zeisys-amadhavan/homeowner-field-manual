# The Homeowner Field Manual — 150 Aha-Page Concept (Improved)

**Working Positioning**  
**Title:** The Homeowner Field Manual  
**Subtitle:** A Visual Guide to the Hidden Systems, Documents, Risks, and Repairs Behind Homeownership  

This is not a generic DIY book. It is a visual ownership manual for ordinary homeowners. Each page must reveal something hidden: a system, risk, document, cost, decision boundary, failure path, or leverage point.

## Core Doctrine  
**Know it. Prove it. Maintain it. Fix it. Defend it.**  
A homeowner is not merely living in a building. A homeowner is managing:  
* A shelter  
* A machine  
* A legal asset  
* A financial asset  
* A risk object  
* A maintenance system  
* A documentation system  

## Page Test  
Every page must pass the “Aha Test”:  
Would a normal homeowner say, “Darn, I didn’t know that”?  
* **Weak page:** “Change your HVAC filter.”  
* **Strong page:** “A dirty filter can freeze the AC coil and mimic an expensive refrigerant problem.”  
* **Weak page:** “Keep receipts.”  
* **Strong page:** “A receipt can become proof for warranty, insurance, resale, HOA approval, or contractor dispute.”

## Book-Level Visual Rules
* Every page should have one dominant visual, not a collage of tiny unrelated images.  
* Every page should reveal a hidden relationship: path, sequence, hierarchy, boundary, timeline, or failure chain.  
* Every page should include a clear homeowner-safe boundary.  
* Every repair/troubleshooting page should integrate part logic: part name, model/photo needed, compatibility warning, and stop rule.  
* Every legal/admin/HOA/insurance page should avoid state-specific legal advice unless localized later; focus on documents, deadlines, proof, and questions to ask.  
* Visual style should match the Winching Field Manual: compact, diagram-heavy, field-usable, high-glance clarity.  
* **New consistency rule:** Use consistent color-coding for paths (water=blue, air/green, electric/red, load/black, heat/orange, pest/yellow) across all pages. Every visual must include a small “5 Lenses” legend corner and a “Know it. Prove it. Maintain it. Fix it. Defend it.” doctrine icon where relevant.  

## Suggested Recurring Page Modules
Use these recurring boxes across the manual (now with clearer templates for implementation):  
* **Aha:** one-line hidden truth (visceral homeowner “Darn, I didn’t know that”).  
* **Map:** what system/path/document this belongs to (color-coded).  
* **Check:** what the homeowner can safely verify (step-by-step).  
* **Bring This:** photos, model numbers, measurements, documents, or proof needed.  
* **Stop If:** clear boundary for calling a professional, utility, HOA, insurer, or emergency service (red-box emphasis).  
* **Record It:** what evidence to keep after the issue is handled.  
* **The Leverage Point ($10 vs. $1,000 Rule):** the specific cost asymmetry on a page to create an instant Aha (e.g., a $15 downspout extender prevents a $15,000 foundation repair).
